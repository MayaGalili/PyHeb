# 🐍 פיתון בעברית - מדריך למתחילים

ברוכים הבאים לעולם התכנות בעברית! כאן תלמדו איך לכתוב תוכניות מחשב באמצעות פונקציות בעברית.

## 📚 מה זה פיתון בעברית?

פיתון בעברית הוא אוסף של פונקציות שמאפשרות לכם לכתוב קוד מחשב בעברית במקום באנגלית. זה עוזר לילדים שמתחילים ללמוד תכנות להבין מה הקוד עושה!

## 🚀 איך להתחיל?

### 1. התקנה
```bash
# הורידו את הקבצים לתיקייה
# הרצו את הבדיקות כדי לוודא שהכל עובד
python בדיקות.py
```

### 2. ייבוא הפונקציות
```python
from פעולות import *
from בולאנים import נכון, לא_נכון
```

## 🔢 פונקציות חשבון

### חישובים בסיסיים
```python
# חיבור
תוצאה = חיבור(5, 3)  # 8

# חיסור
תוצאה = חיסור(10, 4)  # 6

# כפל
תוצאה = כפל(4, 5)  # 20

# חילוק
תוצאה = חילוק(15, 3)  # 5.0

# חזקה
תוצאה = חזקה(2, 3)  # 8

# שורש
תוצאה = שורש(25)  # 5.0
```

### עיגול מספרים
```python
# עיגול למספר שלם
מספר = עגל(3.7)  # 4

# עיגול לספרות אחרי הנקודה
מספר = עגל(3.14159, 2)  # 3.14
```

## 📝 עבודה עם מחרוזות (טקסט)

### פונקציות מחרוזות בסיסיות
```python
# אורך מחרוזת
אורך_טקסט = אורך("שלום")  # 4

# הפיכת מחרוזת
טקסט_הפוך = הפוך("שלום")  # "םולש"

# חיבור מחרוזות
טקסט_חדש = חיבור_מחרוזות("שלום ", "עולם")  # "שלום עולם"

# חיתוך מחרוזת
חלק = חתוך("שלום עולם", 0, 4)  # "שלום"

# חיפוש במחרוזת
נמצא = מצא("שלום עולם", "עולם")  # True
```

## 📋 עבודה עם רשימות

### יצירה וניהול רשימות
```python
# יצירת רשימה ריקה
רשימה_שלי = רשימה_ריקה()

# הוספת ערכים לרשימה
הוסף_לרשימה(רשימה_שלי, "תפוח")
הוסף_לרשימה(רשימה_שלי, "בננה")
הוסף_לרשימה(רשימה_שלי, "תפוז")

# אורך הרשימה
כמה_ערכים = אורך_רשימה(רשימה_שלי)  # 3

# הסרת ערך מהרשימה
הסר_מרשימה(רשימה_שלי, "בננה")
```

### המרה בין מחרוזות לרשימות
```python
# מחרוזת לרשימה
רשימה = רשימה_ממחרוזת("אחד שניים שלושה")
# תוצאה: ["אחד", "שניים", "שלושה"]

# רשימה למחרוזת
מחרוזת = מחרוזת_מרשימה(["א", "ב", "ג"])
# תוצאה: "א ב ג"
```

## 🎯 תנאים ולולאות

### פונקציית אם_אז

## 🎮 דוגמאות משחקים

### משחק ניחוש מספר
```python
from פעולות import *
from בולאנים import נכון, לא_נכון

# משחק ניחוש מספר
מספר_סודי = מספר_אקראי(1, 10)
הדפס("נחש מספר בין 1 ל-10!")

ניחוש = 7
הדפס(f"הניחוש שלך: {ניחוש}")

אם_אז(שווה(ניחוש, מספר_סודי), "🎉 נכון! זכית!", הדפס)
אם_אז(גדול_מ(ניחוש, מספר_סודי), "המספר קטן יותר", הדפס)
אם_אז(קטן_מ(ניחוש, מספר_סודי), "המספר גדול יותר", הדפס)

הדפס(f"המספר הסודי היה: {מספר_סודי}")
```

### מחשבון פשוט
```python
# מחשבון פשוט
def מחשבון():
    הדפס("=== מחשבון פשוט ===")
    
    מספר1 = 10
    מספר2 = 5
    
    הדפס(f"{מספר1} + {מספר2} = {חיבור(מספר1, מספר2)}")
    הדפס(f"{מספר1} - {מספר2} = {חיסור(מספר1, מספר2)}")
    הדפס(f"{מספר1} × {מספר2} = {כפל(מספר1, מספר2)}")
    הדפס(f"{מספר1} ÷ {מספר2} = {חילוק(מספר1, מספר2)}")

מחשבון()
```

### משחק מילים
```python
# משחק הפיכת מילים
def משחק_מילים():
    מילה = "שלום"
    הדפס(f"המילה המקורית: {מילה}")
    
    מילה_הפוכה = הפוך(מילה)
    הדפס(f"המילה ההפוכה: {מילה_הפוכה}")
    
    אם_אז(שווה(מילה, הפוך(מילה_הפוכה)), "הפיכה עבדה!", הדפס)

משחק_מילים()
```

## 🎯 טיפים למתחילים

### 1. התחילו בקטן
- התחילו עם פונקציות פשוטות כמו `הדפס` ו-`חיבור`
- נסו לכתוב תוכניות קצרות

### 2. השתמשו בהדפסה
- השתמשו ב-`הדפס` כדי לראות מה קורה בקוד שלכם
- זה עוזר להבין איך הקוד עובד

### 3. נסו דברים חדשים
- אל תפחדו לטעות!
- נסו לשנות מספרים ומילים בקוד

### 4. כתבו הערות
```python
# זה הערה - היא עוזרת להבין מה הקוד עושה
הדפס("שלום עולם")  # זה מדפיס הודעה
```
